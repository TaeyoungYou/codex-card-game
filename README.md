# codex-card-game

Browser-first starter project for a 4 x 4 memory card game prototype.

## Structure

- `index.html`: browser entry point
- `src/`: memory-game logic, UI rendering, and styles
- `tests/`: placeholder for automated tests
- `assets/`: static assets
- `docs/`: project notes

## Run

Open `index.html` in a browser to play the current prototype.
