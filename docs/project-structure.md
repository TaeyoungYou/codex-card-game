# Project Structure

This project starts as a browser-first 4 x 4 memory card game prototype with no build step.

## Directories

- `src/game/`: card setup, flip rules, and turn resolution
- `src/ui/`: DOM rendering helpers
- `src/styles/`: shared styles for the browser client
- `tests/`: placeholder for gameplay and UI tests
- `assets/`: static assets such as card art and sound effects

## Next Steps

1. Add a score timer and best-record tracking.
2. Add card themes or difficulty settings beyond 4 x 4.
3. Add a lightweight dev server or build tool once the prototype needs bundling.
